## 1. Функции

Функции

```python
class main(): # группа функций
  def __main__(): # функция, которая имеет строки, и может быть исполнена при её вызове (__main__())
    print("Привет!") # print - это вывести текст в консоль
    
    b = input("Как у тебя дела\n> ") # input - поле, где нужно ввести значение. b - это её название, ниже будет пример как с помощью названия функции, можно узнать её значение
    
    if b.lower()=='нормально': == lower() - это функция, которая переводит текст "НорМально" в "нормально". upper() - делает из него "НОРМАЛЬНО"
      # if - это условие
      print("Хорошего тебе дня!")
    else: # else - окончание условия, если человек ввёл другой текст(пример)
      print("Удачи!")
      
__main__() # функция def __main__():
```

## 2. Список

```
"""
Смотрите, это большой комментарий
"""

import random # библиотека для рандомных значений

список = ['привет', 'как дела'] # список, и да, питон поддерживает название функций на русском

print(random.choice(список)) # выводит рандомное значение из списка "список"
```

Также список поддерживает не только обычные значения как """привет""", "привет", 'привет', '''привет''', но и другие как:

```
import random
from random import randint as ii # импортирует из random функцию с другим названием

iii = [ii(2,5), ii(1,4)] # ii(2,5) - ii это я уже объяснил, (2,5) - это рандомное число от 2 до 5.
print(random.choice(iii))
```

## 3. global

```
x = 0

print(f"Ваши очки: {x}") # f"" - это "" с кодом через {}

def rewarn_score(z): # z - это значение, rewarn_score(7)
  global x 
  x = int(x) + int(z) # переводит string (обычный текст) в int (число)
  
def lose_score(z):
  global x
  x = int(x) - int(z)
```
